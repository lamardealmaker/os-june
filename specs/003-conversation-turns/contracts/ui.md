# UI Contract: Conversation Turns

## Transcription Tab

- Displays source transcript rows in chronological turn order.
- Shows `Microphone` and `System` labels for turn source.
- Shows a compact time range when turn timing is available.
- Keeps long transcripts scrollable within the editor panel.
- Falls back to the existing raw transcript display for older notes without source transcript rows.

## Recorder UI

- No new realtime transcript surface is added.
- Existing source mode and recording controls remain unchanged.
